## How to Run Your Framework with OmniDaemon

### Example 1: Running LangGraph with OmniDaemon

**Your LangGraph code** (stays the same):
```python
# my_langgraph_agent/agent.py
from langgraph.graph import StateGraph

def my_langgraph_workflow(message: dict):
    # Your LangGraph logic here
    graph = StateGraph(...)
    result = graph.invoke(message["content"])
    return {"result": result}
```

**Run it with OmniDaemon**:
```python
# agent_runner.py
from omnidaemon import OmniDaemonSDK, AgentConfig
from omnidaemon.supervisor import create_supervisor_from_directory

sdk = OmniDaemonSDK()

# Create supervisor for your LangGraph agent
supervisor = await create_supervisor_from_directory(
    agent_name="langgraph_agent",
    agent_dir="./my_langgraph_agent",
    callback_function="agent.my_langgraph_workflow"
)

# Register with event-driven runtime
await sdk.register_agent(
    agent_config=AgentConfig(topic="tasks.langgraph"),
    callback=supervisor.handle_event
)

await sdk.start()
```

**What you get**:
- âœ… Your LangGraph agent runs in isolated process
- âœ… Auto-restart if it crashes
- âœ… Event-driven (publish tasks to `tasks.langgraph` topic)
- âœ… DLQ for failed tasks
- âœ… Built-in monitoring

### Example 2: Running CrewAI with OmniDaemon

**Your CrewAI code** (stays the same):
```python
# my_crewai_agent/agent.py
from crewai import Agent, Task, Crew

def my_crewai_workflow(message: dict):
    # Your CrewAI logic
    agent = Agent(role="Researcher", ...)
    task = Task(description=message["content"], ...)
    crew = Crew(agents=[agent], tasks=[task])
    result = crew.kickoff()
    return {"result": result}
```

**Run it with OmniDaemon** (same pattern):
```python
supervisor = await create_supervisor_from_directory(
    agent_name="crewai_agent",

    callback_function="agent.my_crewai_workflow"
)

await sdk.register_agent(
    agent_config=AgentConfig(topic="tasks.crewai"),
    callback=supervisor.handle_event
)
```

## Pluggable Backends (No Vendor Lock-In)

**The Simple Truth**: Change backends via environment variables, **zero code changes**.

```bash
# Event Bus - Swap anytime
EVENT_BUS_TYPE=redis_stream + REDIS_URL=...         # Current (production-ready)
EVENT_BUS_TYPE=kafka + KAFKA_SERVERS=...            # Coming Q1 2025
EVENT_BUS_TYPE=rabbitmq + RABBITMQ_URL=...          # Coming Q2 2025

# Storage - Swap anytime
STORAGE_BACKEND=redis + REDIS_URL=...               # Current (production-ready)
STORAGE_BACKEND=postgresql + POSTGRES_URL=...       # Coming Q1 2025
STORAGE_BACKEND=mongodb + MONGODB_URI=...           # Coming Q2 2025
```

**Your agent code never changes. Just update env vars.**
